日本語入力方式SKKのための辞書群
動作確認はCorvusSKKで行なっている

仕様は"specification.txt"

以下 各辞書の説明

bamboo-copter.skk
    竹とんぼ

horticulture.skk
    園芸

idiom.skk
    慣用句
    文脈を考慮した変換ができないSKKの欠点を補うことができる

box-drawing-character.skk
    罫線素片

lisp.skk
    corvusskk用のプログラム実行変換

succulents

        正確性はまだ不十分

    succulents/genus-name.skk
        属名

    succulents/japanese-name.skk

    succulents/scientific-name.skk
        学名

    succulents/term.skk
